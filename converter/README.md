# GSGI 格式转换器 / GSGI Format Converter

**计划中** — DXF ↔ GSGI 双向转换工具（Python + ezdxf）。
**Planned** — DXF ↔ GSGI bidirectional conversion tool (Python + ezdxf).

预期功能 / Planned features：
- `gsgi_cli.py import drawing.dxf -o output.gsgi`
- `gsgi_cli.py export input.gsgi -o output.dxf`
- `gsgi_cli.py validate input.gsgi`
