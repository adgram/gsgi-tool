## 工具操作流程

- 综述：鼠标有两种状态
    1. 鼠标为选择状态，禁用捕捉
    2. 鼠标为绘图状态，根据用户操作，可启用捕捉

- 移动1：选中物体，点击移动，鼠标变为绘图状态，提示点击移动参照点，用户点击移动基点，再点击移动终点，完成移动。
- 移动2：点击移动，鼠标变为选择状态，提示选择实体，[进入选择模式，]选中实体后，回车退出选择模式，鼠标变为绘图状态，提示点击移动参照点，用户点击移动基点，再点击移动终点，完成移动。

- 选择模式：点击实体即选中，再点击其他实体取消原先选中的，改选新的。若按住shift，可以加选，减选。支持左右框选。
- 复制和移动相似；

- 旋转1：选中物体，点击旋转，鼠标变为绘图状态，提示点击旋转参照点，用户点击旋转基点，提示输入旋转角度，用户输入旋转角度后，回车，完成旋转。
- 旋转2：点击旋转，鼠标变为选择状态，提示选择实体，[进入选择模式，]选中实体后，回车退出选择模式，鼠标变为绘图状态，提示点击旋转参照点，用户点击旋转基点，提示输入旋转角度，用户输入旋转角度后，回车，完成旋转。
- 对称1：选中物体，点击对称，鼠标变为绘图状态，提示点击对称参照点1，点击第一个点，提示点击对称参照点2，点击第二个点2，完成对称。
- 对称2：点击对称，鼠标变为选择状态，提示选择实体，[进入选择模式，]选中实体后，回车退出选择模式，鼠标变为绘图状态，提示点击对称参照点1，点击第一个点，提示点击对称参照点2，点击第二个点2，完成对称。
