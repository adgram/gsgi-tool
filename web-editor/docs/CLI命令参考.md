# CLI 命令参考

在命令行输入框中输入命令，支持 Tab 自动补全。按 Enter 执行。

---

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Escape` | 取消当前绘制 / 关闭右键菜单 / 取消选择 / 切回选择工具 |
| `Ctrl+O` | 打开文件 |
| `Ctrl+S` | 保存文件 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` / `Ctrl+Y` | 重做 |
| `Home` | 缩放到全图 |
| `Ctrl+Shift+P` | 恢复上一次选择 |
| `F7` | 切换栅格 |
| `F9` | 切换捕捉 |
| `Tab` | 命令行自动补全 |
| `↑` / `↓` | 命令行历史浏览 |

---

## 文档查询

| 命令 | 用法 | 说明 |
|------|------|------|
| `ls` | `ls [类型]` | 列出实体，可选按类型过滤（如 `ls line`） |
| `count` | `count [类型]` | 按类型统计实体数量 |
| `info` | `info [ID\|编号]` | 显示指定实体的全部属性；不指定时显示当前选中的所有实体 |

## 选择操作

| 命令 | 用法 | 说明 |
|------|------|------|
| `select` | `select <ID\|编号>` | 按 ID（如 `L1`）或序号选择实体 |
| `deselect` | `deselect [ID\|编号]` | 取消选择，不指定则取消全部 |

## 文档编辑

| 命令 | 用法 | 说明 |
|------|------|------|
| `set` | `set [ID\|编号] <属性名> <值>` | 设置实体属性；不指定 ID 时设置所有选中的实体 |
| `delete` | `delete [ID\|编号]` | 删除指定实体；不指定时删除所有选中的实体 |
| `move` | `move [ID\|编号] <dx> <dy>` | 移动实体；不指定 ID 时移动所有选中的实体 |
| `clear` | `clear` | 清空全部实体（需确认） |

## 创建实体

| 命令 | 用法 | 说明 |
|------|------|------|
| `point` | `point [x y]` | 不带参时激活画点工具；带参时在 (x,y) 处直接创建点 |
| `line` | `line [x1 y1 x2 y2]` | 不带参时激活画线工具；带参时直接创建直线 |
| `circle` | `circle [cx cy r]` | 不带参时激活画圆工具；带参时直接创建圆 |

## 视图控制

| 命令 | 用法 | 说明 |
|------|------|------|
| `zoom` / `z` | `zoom [in\|out]` | 不带参数缩放至全图 |
| `grid` | `grid [on\|off]` | 切换栅格显示 |
| `snap` | `snap [on\|off]` | 切换栅格捕捉 |

## 图层管理

| 命令 | 用法 | 说明 |
|------|------|------|
| `layer` | `layer list` | 列出所有图层 |
| `layer` | `layer add <名称>` | 新建图层 |
| `layer` | `layer delete <名称>` | 删除图层（不能删除默认图层 0） |

## 文件操作

| 命令 | 用法 | 说明 |
|------|------|------|
| `new` | `new` | 新建文档 |
| `open` | `open` | 打开文件 |
| `save` | `save` | 保存文件 |

## 其他

| 命令 | 用法 | 说明 |
|------|------|------|
| `help` | `help [命令名]` | 不带参数列出所有命令 |
| `echo` | `echo <文字...>` | 将文字回显到提示栏（测试用） |
